Wat is Bootstrap

Front-end development

De ontwikkeling van de grafische gebruikersinterfaces (UI) van een website, door het gebruik van HTML, CSS en JavaScript. Hierdoor kunnen gebruikers de website bekijken en ermee werken zonder enige kennis

Gratis & open-source

Open-source software is software waarvan de broncode (sources) is gepubliceerd en vrij beschikbaar is voor iedereen. Iedereen kan en mag deze software kopiÃ«ren, aanpassen, aanvullen en verspreiden zonder kosten aan auteursrechten en of toeslagen.

CSS framework

Met een CSS framework heeft de gebruiker een compleet CSS stylesheet, en hoeft hij alleen nog maar de HTML te coderen met de juiste classes, structuur, en IDs om een webpagina op te zetten. Het framework heeft al klassen ingebouwd voor veel voorkomende website-elementen - voettekst, slider, navigatiebalk, hamburger menu, kolom-gebaseerde lay-outs, enz.

Responsief & mobile-first

Een responsive website is reactief - het ontwerp beweegt vloeiend mee met de apparaten. Bij een mobile-first website wordt de mobiele website samen met de desktopsite gepland en ontworpen, waarbij proactief wijzigingen in het totale ontwerp worden aangebracht om ervoor te zorgen dat de mobiele ervaring net zo goed is als de desktopervaring.
